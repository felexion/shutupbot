# Shut Up Bot

A lightweight chaos generator for your server.

## Features

- **10% chance** to reply to a message with “shut up”.
- **1% chance** to reply with an angrier version.
- Triggered when:
  - The bot is mentioned
  - A user replies to one of the bot’s messages
  - A user runs `/shut`
